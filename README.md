## Установка

Для установки необходимо:
* Скачать файл files.ospx из раздела [releases](https://github.com/khorevaa/files/releases)
* Воспользоваться командой:

```
opm install -f <ПутьКФайлу>
```
или установить с хаба пакетов

```
opm install files
```
